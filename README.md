# AI Operating System Workspace

## Обзор

Это масштабируемая AI-операционная система для совместной работы AI-агентов, Codex и Cloud Code в VS Code. Платформа построена для production-ready развертывания с разделением на рабочий (belberry) и личный (personal) контуры.

## Архитектура

### Контуры
- **shared**: Переиспользуемые компоненты, шаблоны и инфраструктура
- **belberry**: Рабочий контур для бизнес-проектов
- **personal**: Личный контур для экспериментов и личных проектов
- **archive**: Архив завершенных проектов

### Компоненты каждого контура
- **agents**: AI-агенты и их конфигурации
- **prompts**: Системные промпты и шаблоны
- **docs**: Документация и архитектурные решения
- **integrations**: Интеграции с внешними сервисами (Bitrix24, Google API, etc.)
- **apps**: Веб-сервисы и приложения
- **data**: Данные и конфигурации
- **logs**: Логи и метрики
- **experiments**: Экспериментальные проекты
- **automation**: Автоматизации и workflows
- **sandbox**: Тестовая среда
- **assets**: Статические ресурсы

## Технологии

- **Node.js/TypeScript**: Основной стек разработки
- **OpenAI SDK**: Интеграция с AI-моделями
- **Telegram Bots API**: Боты для коммуникаций
- **Bitrix24 API**: Интеграция с CRM
- **Docker**: Контейнеризация
- **Git**: Версионирование

## Начало работы

1. Установите зависимости: `npm install`
2. Настройте переменные окружения: скопируйте `.env.example` в `.env`
3. Запустите development сервер: `npm run dev`

## Разработка

- Используйте shared компоненты для переиспользования
- Соблюдайте разделение контуров
- Документируйте все изменения
- Тестируйте в sandbox перед production
