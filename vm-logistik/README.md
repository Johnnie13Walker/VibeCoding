# ВМ-Логистик — внедрение ИИ

Клиент Жени. С **10.06.2026** Женя — руководитель проекта по внедрению ИИ в ВМ-Логистик
(крупная транспортно-логистическая компания, ~6 млрд ₽/год, 55 человек, ФОТ 57–58%).

> Канонический контекст, цели, контуры, стейкхолдеры и техландшафт — в Obsidian:
> `Cloudbot-Vault/09-Projects/vm-logistik/_README.md` и `STATUS.md`.
> Эта папка — для кода/скриптов/артефактов проекта.

## Миссия

Превратить ИИ в деньги (новая выручка) и экономию (сокращение ФОТ логистов), доведя ядро
до закрытия с минимальным человеческим трудом.

## Контуры (кратко)

- **A. Электронный логист** — парсинг заявок + подбор перевозчика из базы 75k (1С) + голос/мессенджер-коммуникация + согласование цены (floor ~15%). Отдельное подразделение.
- **B. Больше заявок + торги** — авто-участие в тендерах (сейчас 3 из 400 вручную), дашборд присутствия на площадках.
- **C. Командный центр** — P&L по офисам, IP-телефония + запись + транскрибация + оценка (аналог Belberry GSD).
- **D. Продажи** — база лидов + доступ к CRM конкурентов, забор тёплых лидов.
- **E. ЭТрН** — электронные ТТН с сентября 2026.

## Техландшафт

- **1С** — ядро логистики + база перевозчиков. **Bitrix24** — слабое внедрение (рабочая группа ЭТрН, чаты).
- **ATI.SU** — биржа, «единственное окно» логиста. Торговые площадки — с API.
