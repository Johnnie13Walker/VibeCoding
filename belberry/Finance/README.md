# Finance

Рабочий финансовый контур Belberry.

## Структура

- `source/raw/` — первичные старые файлы, перенесённые с Desktop без удаления исходников.
- `models/` — рабочие финмодели.
- `reports/` — отчёты и выгрузки.
- `scripts/` — скрипты обработки и сверки.
- `docs/` — методология, правила сверки, промты.
- `data/` — нормализованные данные для автоматизации.

## Правило безопасности

Сырые финансовые Excel/CSV в `source/raw/` не коммитятся в git. Если нужно зафиксировать структуру или выводы, добавлять документацию, скрипты или нормализованные обезличенные данные отдельно.
