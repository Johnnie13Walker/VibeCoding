## 1. Контекст и цель

Маркетинг-кит для холодной отправки и B2B-встреч. Цель — продать **комплексное digital-агентство на длинной дистанции**, не выглядеть как очередное агентство.

Кит не каталог услуг. Это **нарратив из 4 актов**:
1. Конфликт (что сломалось в digital)
2. Наша модель (как мы устроены)
3. Доказательства (кейсы, клиенты, команда)
4. К решению (страхи → ответ, прайс, финал)

Текущий `.pptx` — функциональный черновик, **выверенный по смыслам и геометрии**. Задача дизайнера — поднять визуально до уровня premium B2B-кита.

## 5. Фирменный стиль

### 5.1 Палитра

| Цвет | HEX | Где используется |
|---|---|---|
| Dark | #1F1F22 | Фон тёмных слайдов |
| Light | #EBEBEE | Фон светлых слайдов |
| Blue (primary accent) | #2B7CE9 | Главный акцент — метрики, CTA, hero-цифры |
| Purple (GEO accent) | #6B4FE9 | GEO как новое направление, выделение |
| Gold | #C99A3D | Награды, премии, hero-карточка NGENIX |
| Gray text | #7A7A85 | Подписи, второстепенный текст |
| Light gray text | #CCCCD0 | Текст на тёмном фоне |
| Red (warning) | #C83E3E | Цена ошибки, негативные индикаторы |
| Deep blue | #1B3F8E | Hero-карточки клиентов |
| Near-black | #22 22 26 | Hero-карточка Главстроя |

### 5.2 Типографика

**Заголовки:** Manrope ExtraBold (Google Fonts) или TT Norms Pro Bold (премиум)  
**Текст:** Inter Regular / Manrope Regular  
**Цифры на статах:** тот же grotesque, weight 800, размер 72–130pt  

Размеры по иерархии:
- H1 (главный заголовок слайда) — 38–46pt
- H2 (подзаголовки в карточках) — 18–24pt
- Body — 12–14pt
- Caption — 9–11pt
- Hero numbers — 72–130pt

### 5.3 Шапка слайда (единый элемент на КАЖДОМ слайде)

- **Левый верхний угол:** «WWW.ACOOLA.TEAM» — 10pt серым
- **Правый верхний угол:** логотип «ACOOLA TEAM» (вертикально-разделённая композиция: ACOOLA крупно 18pt + TEAM мелким 10pt снизу справа)
- **Под логотипом:** номер слайда «N / 26» (9pt серым)

### 5.4 Композиционные приёмы

- **Pill-плашки с эмодзи** на обложке/CTA: «🔥 Прокачаем», «📈 Выведем в топ», «💼 Увеличим прибыль» (эмодзи здесь — фирменный приём, в остальном тексте запрещены)
- **Стрелка ↗** в правом верхнем углу карточек клиентов, услуг — фирменный знак интерактивности
- **Скруглённые карточки** radius 16–20px, без тяжёлых теней
- **Тонкие разделительные линии** между ячейками сеток

### 5.5 Запрещено

- Стоковые иконки и иллюстрации (handshake, лампочки, мишени)
- 3D-иллюстрации в стиле «корпоративный минимализм 2018»
- Эмодзи в основном тексте слайдов (кроме фирменных pill-плашек на обложке/CTA)
- Чистый чёрный #000000 — только тёплый тёмно-серый #1F1F22

## 7. Топ-7 ключевых слайдов и их специфика

### S1 Обложка-провокация
- Тёмный фон, **массивная типографика**: «Кто-то обещает топ-10. Мы — нет.»
- Pill-плашки «🔥 Прокачаем» / «📈 Выведем в топ» / «💼 Увеличим прибыль»
- Плашка персонализации справа: `{COMPANY_NAME}`, `{DOMAIN}`, `{N точек роста}` — заполняется маркетологом под каждого клиента
- **Референс:** Linear changelog cover, Vercel Ship 2024

### S2 Манифест (4 анти-обещания)
- Светлый фон, лестница из 4 утверждений
- Каждое: «Мы не X. Мы Y.» Эффект — типографическая стена
- **Референс:** Basecamp handbook, 37signals manifesto

### S10 GEO — кульминация акта 2
- Тёмный + **электрик-фиолетовый** акцент (отличается от синего основной палитры)
- Слева — мокап ответа AI-ассистента (заменить наш placeholder на реальный скриншот ChatGPT/Perplexity)
- Справа — таймлайн 5 шагов работы
- Цена «от 90 000 ₽/мес» крупно
- **Референс:** Linear product pages, Vercel AI products

### S11–14 Кейсы (TEO, wwerb, Kenner, ORM)
- Двухколоночный: **слева** — история в 3 абзаца, **справа** — визуальный артефакт (график падения CPL для wwerb, ROI 140% для Kenner, мокап-скриншот для TEO, bar chart 29%→7% для ORM)
- **Референс:** Linear customers, Vercel customer cards, Notion customer stories

### S15 25 клиентов · сетка 5×5
- Тёмный фон. Слева подписи отраслей серым, справа — 5 ячеек с **логотипами клиентов** (не текст)
- В правом верхнем углу каждой ячейки — стрелка ↗ серым
- Логотипы клиентов — **монохромные**, единой высоты
- **Референс:** фирменный слайд «Наши клиенты» Acoola (главный референс) + Sequoia our-companies

### S22 5 страхов → наш ответ
- Светлый фон, 5 строк формата «БОЯЛИСЬ … → ПРОПИСАНО В ДОГОВОРЕ …»
- Слева страх курсивом, справа жирно
- Стрелка → синяя в центре
- **Референс:** Basecamp "How we work", any "myths vs facts" B2B template

### S25 Финал A vs B
- Тёмный, две большие карточки. Слева — серая (вариант А = 5 подрядчиков), справа — синяя (вариант B = 1 команда)
- Контактная плашка снизу
- **Референс:** any B2B comparison deck
