## 0. Дизайн-направление и фирменный стиль Acoola

### 0.1 Палитра

**Тёмные слайды** — для логотипов клиентов, кейсов, сильных контентных блоков:
- Фон: **#1F1F22** (тёплый тёмно-серый, не чистый чёрный)
- Основной текст: белый
- Подписи и второстепенный текст: серый ~**#7A7A85**
- Разделители: rgba(255,255,255,0.08)

**Светлые слайды** — для цифр о компании, наград, статистики, CTA:
- Фон: **#EBEBEE** (светло-серый, не белый)
- Основной текст: чёрный
- Акцент-цвет для цифр-метрик («50+», «70+», «3,5», «95%»): **синий ~#2B7CE9**
- Карточки на белом #FFFFFF поверх светло-серого

**Карточки наград** — каждая со своей цветовой идентичностью (тёмная / синяя-электрик / красная / золотая / тёмная с призмой). Не плоские: с фактурой, металликом, градиентами.

### 0.2 Типографика

- **Заголовки H1:** плотный grotesque (Manrope ExtraBold / TT Norms Pro Bold / Suisse Bold), 56–72pt, weight 800
- **Основной текст:** Inter Regular / Manrope Regular, 16–18pt
- **Цифры на статах:** тот же grotesque, синий акцент-цвет, 72–96pt, weight 800
- **Подписи под цифрами:** серым, 14–16pt, weight 400

### 0.3 Шапка слайда — единый фирменный элемент (на КАЖДОМ слайде)

**Acoola.Team превращает сайт из визитки в системный канал продаж: разрабатываем, продвигаем и поддерживаем — одной командой, на дистанции, с измеримым ростом обращений и выручки. Нам доверяют Сбер, БыстроБанк, Pioneer, Главстрой и NGENIX.**

### 1.7 Брендвойс — памятка для дизайнера и редакторов

**Тон.** Уверенный, деловой, плотный. Без панибратства, без пафоса, без канцелярита. Допустим лёгкий маркетинговый драйв в pill-плашках («Прокачаем», «Выведем в топ», «Увеличим прибыль») — это фирменный приём.

**4 принципа письма:**

1. **Цифры впереди эпитетов.** Не «много кейсов», а «70+ проектов в работе ежемесячно».
2. **Доказательства рядом с утверждениями.** Сказали «качество» — приложили «95% довольных клиентов».
3. **Анти-обещания сильнее обещаний.** «Не обещаем топ — показываем рост» работает лучше, чем «гарантируем результат».
4. **Каждое УТП снимает возражение клиента.** Не хвалим себя — отвечаем на «а что, если…».

**Стоп-лист (если встречается — переписать):**
- «Команда профессионалов»
