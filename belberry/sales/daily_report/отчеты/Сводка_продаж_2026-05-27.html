Технический долг / ограничения

  • Товарные позиции (productrows) пустые. Запрос crm.item.productrow.list по 51 активной сделке вернул 0 заполненных карточек. Сумма OPPORTUNITY у 26 сделок задана вручную, у 25 — равна нулю. Это блокирует автоматическую сверку «сумма сделки = Σ(позиции КП)».
  • Wazzup в активности не попадает. За день 0 записей с PROVIDER_ID = WAZZUP в crm.activity.list. Все клиентские диалоги доступны только через crm.timeline.comment.list. Отчёт собирает оба источника.
  • Wazzup-комментарии в timeline лежат с AUTHOR_ID=2358 (технический пользователь интеграции Тартышева). Реальный автор и факт «итогов встречи клиенту» нужно парсить из тела COMMENT по маркеру «Имя Фамилия:». Без этого автоматическая проверка «есть ли итоги» даёт ложноотрицательные срабатывания.
  • Аватары пользователей из Битрикс CDN без авторизации не доступны → встроены base64 (140px по длинной стороне).
  • Транскрипты встреч скачиваются только через REST с auth-токеном — для архивации стоит делать локальную копию txt в день встречи.
