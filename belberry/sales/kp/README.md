# КП Belberry — движок коммерческих предложений

Папка для сборки коммерческих предложений Belberry в фирменном pitch-стиле
(тёмно-фиолетовый титул, светлые контентные слайды, карточки с большими цифрами,
красные дипломы Рейтинга Рунета на финале).

Подход — **образец-копия**: берём готовый эталон `clients/gutaclinic/`, копируем
в новую папку клиента, правим текст/цены прямо в `kp.html`, собираем командой.

## Как сделать новое КП

```bash
cd belberry/sales/kp

# 1. Скопировать эталон
cp -r clients/gutaclinic clients/<новый-клиент>

# 2. Отредактировать contents — текст, цены, услуги, прогнозы
#    (правится прямо в HTML, разметка блоков описана в BLOCKS.md)
$EDITOR clients/<новый-клиент>/kp.html

# 3. Собрать PDF + PPTX
bash build.sh clients/<новый-клиент> "КП Belberry — <клиент>"
```

На выходе в папке клиента: `<Имя>.pdf`, `<Имя>.pptx`, `slides_png/`.

## Структура

```
kp/
├── README.md          # этот файл
├── BLOCKS.md          # карта слайдов и CSS-классов (что где править)
├── build.sh           # HTML → PDF → PNG → PPTX
└── clients/
    └── gutaclinic/    # эталонный кейс (27.05.2026)
        ├── kp.html                       # исходник презентации (11 слайдов)
        ├── КП Belberry — gutaclinic.ru.pdf
        ├── КП Belberry — gutaclinic.ru.pptx
        ├── КП Belberry — gutaclinic.ru.docx   # Word-версия (полный текст)
        ├── kp_obshchee_source.md         # markdown-исходник Word-версии
        └── audit_dela_24412.md           # аудит сделки (внутренний)
```

## Требования (macOS)

| Инструмент | Зачем | Установка |
|---|---|---|
| Google Chrome | рендер HTML→PDF (headless) | уже есть |
| poppler (`pdftoppm`, `pdfinfo`) | PDF→PNG для PPTX | `brew install poppler` |
| `python-pptx` | сборка PPTX из кадров | `pip install python-pptx` |
| pandoc | Word-версия КП из markdown | `brew install pandoc` |

## Важные договорённости (из кейса gutaclinic)

- **Цены из смет всегда без НДС** — в КП добавляем НДС 5% (× 1,05), затем скидку 5%
  при ранней оплате (× 0,95). Две колонки: «с НДС» и «со скидкой».
- **Логика денежного потока единая на 3 слайдах**: 1-й месяц (разовое + первый
  ежемесячный) / 2-й и далее / всего за 12. Не «разово/ежемесячно» по отдельности.
- **Прогнозы — щадящие, с дисклеймером** «оценка на текущий момент, может
  корректироваться». Не давать точных цифр там, где их честно нет (нейровыдача — диапазоны).
- **Должности**: Гордиенко Евгения — РОП (не исполнительный директор; испол. дир. — Качесов Роман).
- **Опыт Belberry — 11 лет (с 2015)**, не «9 лет».
- **PPTX = картинки страниц PDF** (дизайн 1:1, текст не редактируется). Для
  редактируемого PowerPoint нужен отдельный сбор нативными элементами.
