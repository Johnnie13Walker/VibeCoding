# DESIGN.md — дизайн-система деки КП Belberry

> Источник истины для ЛЮБОГО агента/редактора, который трогает `templates/seo-belberry/kp.html`.
> Belberry — медицинский маркетинг. Брендовый источник: belberry.net (лого, цвета сняты 16.06.2026).
> Структурно дека повторяет Acoola (те же маркеры/рендеры), отличие — бренд. Отступление = регрессия.

## Носитель
- Слайд 1280×800, `overflow:hidden`, печать Chrome headless → PDF. Только inline-SVG и data-URI.
- Маркеры `<!--AUTO:*-->`, `{{ПЛЕЙСХОЛДЕРЫ}}`, атрибут `data-lock` — неприкосновенны.
- Акцент-цвета графиков/карточек/болей идут из Python (`set_palette("belberry")`), НЕ из шаблона.

## Цвет
| Токен | Значение | Использование |
|---|---|---|
| accent | `#6B5AF9` | акцент: активные бары, ссылки, метки, цифры, стрелки |
| accent-deep | `#4A3FD0` | низ градиентов |
| indigo | `#2A2367` | тёмный текст-акцент, плашки, сайдбар титула |
| ink-deep | `#141037` | фон тёмных слайдов (титул-сайдбар низ, финал) |
| soft | `#D9D3EE` | «чужие» бары графиков |
| tint | `#ECE9FC` / `#EFECFD` | подложки бейджей, светлые карточки-акценты |
| ink | `#1a1f3a` / `#313131` | основной текст (чистый #000 запрещён) |
| muted | `#717885` / `#9a9eb2` | подписи, источники |
| bg | `#F3F4F9` | фон контент-слайдов |
| Семантика | зелёный `#0a8f5f` · красный `#a13442` · оранжевый `#e88a3b` | хорошо · проблема · дисклеймер |

Соответствие палитре в коде: `PALETTES["belberry"]` в `kp_pipeline.py` (accent/deep/soft).

## Логотип
Inline-SVG Belberry (стетоскоп в хвосте «y», viewBox 0 0 196 65): натуральный (индиго #2A2367 + акцент #6B5AF9) на светлом; белая версия (#2A2367→#FFFFFF) на тёмном (титул-логотип, финал). Текстом «Belberry» не заменять.

## Ритм деки
Светлый титул (индиго-сайдбар слева, крупное имя клиники фиолетовым) → светлые контент-слайды → тёмный индиго-финал с ЭКГ-пульсом. «Немного о Belberry» и «Кому помогаем» — нативные фиолетовые слайды (НЕ запечённые картинки Acoola).

## Тон контента
- Медицинский фрейминг: «пациенты», «клиника», ниши медицины; учёт 323-ФЗ и запрета суперлативов.
- Каждая цифра — из данных клиента (Метрика приоритетнее PR-CY) с источником.
- Запрещены: «гарантируем», англицизмы без нужды, чистый чёрный, синие токены Acoola, цифры-образцы.

## Шапка слайдов (единая концепция)
`.topbar`: `BELBERRY.NET` слева, тёмный логотип справа. Номер — мелкий серый в правом нижнем углу. Контакты — только на финале. Синей линейки/подвалов с дублем лого нет.
