# BLOCKS.md — карта слайдов и CSS-классов КП Belberry

Эталон: `clients/gutaclinic/kp.html` (11 слайдов). Ниже — что за слайд, какие
классы используются, что править под нового клиента.

## Глобальные стили

| Класс | Назначение |
|---|---|
| `.slide` | контейнер слайда 1280×800, flex-column, footer прижат вниз |
| `.slide.title` / `.slide.closing` | тёмно-фиолетовый градиент (титул и финал) |
| `.slide-header h2` + `.pagenum` | заголовок + номер «NN / 11» (правь при добавлении слайдов!) |
| `.slide-rule` | чёрная линейка под подзаголовком |
| `.card` + `.bar` | белая карточка с фиолетовой полоской сверху |
| `.highlight` | фиолетовый блок с 3 колонками крупных цифр |
| `table.belb` | таблица (прогнозы, тоталы); `.future` — фиолетовый акцент, `.totalrow` — итог |
| `.disclaimer` | оранжевый блок «оценка на текущий момент» |
| `table.mediaplan` | календарь-медиаплан (слайд 9) — единая лавандовая гамма |
| `.rr-cert` | красная карточка-диплом Рейтинга Рунета (финал) |

## Слайды эталона

| # | Слайд | Что править под клиента |
|---|---|---|
| 01 | Титул | имя клиента, домен, дата, дедлайн скидки |
| 02 | Состав комплекса | 3 карточки услуг + цены; highlight «1-й мес / 2-й / за 12» |
| 03 | Перенос сайта · детально | этапы/часы (если услуга есть) |
| 04 | ORM + ГЕО · детально | состав ежемесячных работ + цена |
| 05 | Нейровыдача · детально | состав работ + цена |
| 06 | Прогноз ORM (щадящий) | таблица по площадкам + дисклеймер |
| 07 | Прогноз нейровыдачи | диапазоны 3/6/12 + дисклеймер |
| 08 | Сводный тотал | таблица всех сумм + риск без переноса |
| 09 | Календарный план (медиаплан) | услуги×месяцы, 2 итоговые строки (с НДС / со скидкой) |
| 10 | Дорожная карта | сроки, что от вас / что от нас, контакты |
| 11 | О Belberry (финал) | 3 диплома Рейтинга Рунета + полоса фактов |

## Чек-лист при правке нового КП

1. Пересчитать цены: смета (без НДС) × 1,05 (НДС) × 0,95 (скидка). Проверить суммы.
2. Денежный поток (1-й/2-й/12) согласован на слайдах 02, 08, 09.
3. Прогнозы — щадящие + дисклеймер.
4. Обновить `.pagenum` если менял число слайдов (везде `NN / <всего>`).
5. Контакты и должности (Гордиенко — РОП).
6. Дипломы на финале — релевантные продаваемым услугам.
7. Собрать `build.sh`, открыть PDF, проверить что ничего не обрезано
   (частые места: дисклеймеры, highlight внизу — флекс-футер должен прижиматься).

## Сборка

```bash
bash build.sh clients/<клиент> "КП Belberry — <клиент>"
```
