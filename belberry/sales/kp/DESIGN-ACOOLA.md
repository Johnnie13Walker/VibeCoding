# DESIGN.md — дизайн-система деки КП Acoola Team

> Источник истины для ЛЮБОГО агента/редактора, который трогает `templates/seo-acoola/kp.html`
> или создаёт новые слайды/шаблоны бренда Acoola. Извлечено из acoola.team (11.06.2026)
> и боевого шаблона. Отступление от этого файла = регрессия бренда.

## Контекст носителя
- Слайд: **1280×800px**, `overflow:hidden`, печать Chrome headless → PDF (никаких
  внешних запросов, backdrop-filter, анимаций; только inline-SVG и data-URI).
- Дека собирается движком: **маркеры `<!--AUTO:*-->` и `{{ПЛЕЙСХОЛДЕРЫ}} неприкосновенны**,
  как и атрибут `data-lock` и классы из «Компоненты движка» ниже.

| Токен | Значение | Использование |

## Типографика
- **Manrope** (data-URI, 400/500) — весь текст. **Druk Wide Cy** (500) — только короткие
  UPPERCASE-метки через класс `.brand-mark`. Системные фолбэки оставить.
- Шкала: h1-титул 36/66px (вторая строка — имя клиента, 800); h2 слайда **32px/800/−0.03em**;
  подзаголовок 13px muted; текст карточек 12–13px/1.5; подписи и источники 10–11px.
- **Числа-акценты**: kpi 27px/800; pain-banner 52px/900; цифры финала 54px/900;
  «стоимость заявки» 32px/900. Большое число + мелкая подпись — главный приём деки.

## Сетка и воздух
- Паддинг слайда: 44px 64px 26px. Карточки: радиус **12px**, паддинг 18–22px, gap 14–16px.
- Таблицы `.belb`: строки 10–12px паддинга, тонкие разделители, числа `.num` —
  tabular, выравнивание вправо.
- Тонкая линейка `.slide-rule`: 2px, серая с синим сегментом 64px. Не делать жирных чёрных полос.

## Компоненты (использовать существующие, не изобретать)
- `.contact-bar` — синий градиентный подвал 3 колонки (КОНТАКТ/EMAIL/САЙТ).
- `.pain-banner` — красный градиент, `.pb-num` 52px + `.pb-txt`; источник в скобках.
- `.pgroup` (+`.ph red|amber|blue`, `.src`) — колонки «Что мешает».
- Пилюли: радиус 999, 11px/700; бренд-синяя заливка для активной.
- Иконки: **inline-SVG 16–18px, stroke `#3086FB` 2.2px**, на подложке 34px `#eaf2fe`
  радиус 10px. Никаких эмодзи в роли иконок в новых блоках.
- Графики: SVG-бары (высота 20–26px, радиус 5–6px; «свой» — бренд-синий, чужие — `#C9D7EC`,
  подпись значения справа 11–13px/700–800); спарклайн — линия 3px бренд + точки.
- Стекло на синем: `rgba(255,255,255,.10–.14)` + бордер `rgba(255,255,255,.2–.3)`,
  радиус 14px. Без blur.
- Скрин сайта: браузер-рамка (три точки + домен) поверх волн, ширина 400px справа на титуле;
  мини-скрины проблемных страниц — рамка `#EDF1F7`, подпись красным с источником.

## Ритм деки
Тёмно-синий титул → белые контент-слайды → 1–2 акцентных (`#EEF5FF` / синяя карточка
итога) → синий финал. Подряд более 5 одинаково-белых слайдов — плохо.

## Тон контента (связано с движком)
- Каждая цифра — из данных клиента (Метрика приоритетнее PR-CY) с подписанным источником.
- Запрещены: «гарантируем», KPI-обещания, чистый чёрный, англицизмы без нужды,
  цифры-образцы других клиентов.
- Прогнозы — диапазоны + оранжевый дисклеймер.

тёмный `#313131` в подвалах (60×16). Текстовое «ACOOLA TEAM» вместо лого не писать.
