# Personal Workspace

## Обзор

Личный контур AI Operating System для экспериментов, обучения и персональных проектов. Этот контур предназначен для быстрого прототипирования и тестирования новых идей без влияния на production код.

## Структура

- **agents/**: Экспериментальные AI-агенты
- **prompts/**: Промпты для личных задач
- **docs/**: Документация экспериментов
- **integrations/**: Тестовые интеграции
- **apps/**: Персональные приложения
- **data/**: Тестовые данные
- **logs/**: Логи экспериментов
- **experiments/**: Активные эксперименты
- **automation/**: Личные автоматизации
- **sandbox/**: Игровая площадка
- **assets/**: Личные ресурсы

## Цели

1. **Обучение**: Изучение новых AI-технологий
2. **Прототипирование**: Быстрое создание MVP
3. **Эксперименты**: Тестирование гипотез
4. **Инновации**: Разработка новых подходов

## Guidelines

### Development
- **Fast Iteration**: Быстрое прототипирование
- **Experimentation**: Свобода в выборе технологий
- **Learning Focus**: Приоритет на обучении
- **No Production Pressure**: Без требований к reliability

### Best Practices
- **Documentation**: Документировать все эксперименты
- **Version Control**: Все изменения в Git
- **Clean Code**: Даже в экспериментах поддерживать качество
- **Security**: Базовые меры безопасности

## Migration to Production

Когда эксперимент готов к production:
1. **Code Review**: Проверка качества кода
2. **Security Audit**: Проверка безопасности
3. **Testing**: Добавление тестов
4. **Documentation**: Полная документация
5. **Migration**: Перемещение в belberry контур
