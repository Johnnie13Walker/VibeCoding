## Структура листа `Замены`

| Колонка | Содержимое |
|---|---|
| A | Комната / позиция |
| B, C | Текущий вариант (URL+название), цена |
| D, E | Альтернатива (средний сегмент) |
| F | Комментарий (риск + действие) |
| G, H | Премиум-аналог |

Строка 14 — `ИТОГО` с формулами `=SUM(C2:C14)` / `=SUM(E2:E14)` / `=SUM(H2:H14)`.

Шапка серая `#1F2937` заморожена. Ячейки B/C розовый, D/E зелёный, G/H жёлтый. Ссылки через `=HYPERLINK(url; "название")`. Цены — числами.

## Скрипты

| Файл | Что делает |
|---|---|
| `scripts/read_zameny.py` | Читает лист `Замены` (значения + формулы), выводит JSON. Используется для аудита перед правками. |
| `scripts/update_zameny.py` | Эталон batchUpdate для массовой правки ячеек D/E/G/H/F. Текущая версия применила правки 2026-05-10 (исправление 4 битых ссылок). |
| `scripts/update_grohe_essence.py` | Точечная правка G3/G4/F3/F4 для Grohe Essence (URL + артикул скрытой части). |
| `scripts/update_bath_premium.py` | Замена G8/H8/F8 — премиум-ванна на Delice Diapason (литьевой мрамор, после отказа от Bette сталь). |
| `scripts/read_smeta_rooms.py` | Скачивает смету `.xlsx` из Drive, читает листы по жилым комнатам, фильтрует строки со словами сантехники. Использовался для подтверждения, что в спальнях/кабинете сантехники нет. |

## Правила подбора (см. также память Claude)

- **Цвет металла**: брошированная/оружейная сталь (gunmetal/графит матовый). Сверять с заводским цветом.
- **Бренды-якоря**:
  - *Средний*: IDDIS, Omnires, Boheme, Langberger, Ravak, Astra-Form, Эстет, Geberit, Terminus, Omoikiri
  - *Премиум*: Hansgrohe, Grohe, Roca — обязательно дисклеймер «параллельный импорт, заводская гарантия в РФ не действует»
- **Ванна СУ 5.0**: только 180×70, **только литьевой мрамор / каменная масса** (сталь Bette/Kaldewei не использовать)
- **Ссылки**: только конкретные карточки товара, не каталоги. Перед записью обязательно проверить in-stock (`В наличии` / `Купить`). «Под заказ» / «Out of stock» — не годится без явного согласия клиента.
- **Анти-бот**: Ozon/WB JS-challenge ≠ битая ссылка. Не выкидывать без проверки в браузере. Cloudflare-challenge на mosplitka.ru — автопроверка не пройдёт, проверять вручную.
