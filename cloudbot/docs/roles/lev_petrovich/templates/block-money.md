# Архивный денежный deep-dive

## Статус

Файл сохранён как reference после legacy-формата. В канонической ежедневной сводке отдельного блока `Деньги` больше нет: денежные строки уже встроены в `templates/block-funnel.md`.

## Когда этот файл допустим

- для внутренней сверки legacy-формата;
- для миграционного сравнения старого и нового шаблона;
- для отдельного денежного deep-dive по запросу, если он явно помечен как дополнительный аналитический срез, а не как соседний обязательный блок ежедневной сводки.

## Когда использовать нельзя

- нельзя вставлять этот блок рядом с `📈 Воронка в работе` в канонической ежедневной сводке;
- нельзя подменять им основной daily-контракт роли;
- нельзя выдавать его как обязательный пользовательский формат для Telegram-отчёта Льва Петровича.

## Допустимый вспомогательный формат

```text
💸 Денежный deep-dive
• Воронка в работе: <b>{pipeline_amount}</b>
• Высокая вероятность: {high_probability_deals} / <b>{high_probability_amount}</b>
• Под риском: {risk_deals} / <b>{risk_amount}</b>
• Без следующего шага: {without_next_step_deals} / <b>{without_next_step_amount}</b>
• Запущенные без коммуникации > 14 дн.: {stale_deals} / <b>{stale_amount}</b>
• Ушло в отказ за вчера: {lost_yesterday_deals} / <b>{lost_yesterday_amount}</b>
```

## Принцип безопасности

- если денежные данные не подтверждены, писать ограничение по строке или по всему deep-dive, а не подставлять выдуманные нули;
- не добавлять сюда аналитику уровня CFO, маржу, ДДС или управленческий комментарий, который уже должен жить в `Фокусе РОПа`.
