# Блок `Фокус РОПа`

## Назначение

Короткий жёсткий управленческий блок для коммерческого директора. Его задача: не пересказывать весь отчёт, а быстро показать, где потерян контроль, где сегодня можно взять деньги, где РОП должен вмешаться лично и где продажа сорвалась между обязательными шагами.

## Что уже есть в Sales Copilot и не должно дублироваться

- `📈 Воронка в работе` или денежный срез внутри неё уже владеют полными агрегатами по количеству и сумме.
- `🗂 Просрочки по менеджерам` уже владеют полным списком просрочек и примеров по людям.
- `📞 Коммуникации за вчера` уже владеют таблицей активности за день.
- `⚠️ Сделки без следующего шага` и `🚨 Сделки без коммуникации > 14 дн.` уже владеют полным списком проблемных сделок.
- `Фокус РОПа` забирает только отклонение, управленческий приоритет и действие на сегодня.

## Антидубль

- Убирать из старого формата голые агрегаты вроде `+3 без следующего шага`, если тот же смысл уже раскрыт в воронке.
- Не повторять полный список просрочек, а оставлять только короткий управленческий кейс: кто потерял контроль и что нужно закрыть сегодня.
- Не копировать таблицу коммуникаций, а использовать её только как основание для сигнала `тишина`, `нет контакта`, `просадка`.
- Не выводить отдельный подзаголовок `Сделки без следующего шага`: этот список уже живёт в `Sales Copilot`.
- Не повторять одну и ту же сделку сразу в трёх соседних подпунктах без смены угла: для каждой строки должен быть свой управленческий смысл.
- Пустые подпункты скрывать.

## Что должно попасть в блок

- `Где потерян контроль`:
  - максимум `3-4` самых важных сигнала;
  - показывать только потерю управляемости по менеджеру или по группе сделок;
  - основания: нет `следующего шага`, поздняя стадия без движения, нет коммуникации, повторные переносы сроков.
- `💸 Где взять деньги сегодня`:
  - только `2-3` сделки, где деньги ещё живы и есть шанс быстро вернуть движение;
  - на строке должны быть сделка, сумма и одно понятное действие на сегодня.
- `☎️ Где РОПу самому связаться с клиентом`:
  - только `1-2` сделки;
  - основания: большая сумма, долгая тишина, поздняя стадия, менеджер уже не вывез, высокий риск потери.
- `🕳 Упущенные возможности`:
  - только сделки, где продажу не довели до обязательного шага;
  - ключевой провал: нет перехода от брифа к защите КП.
- `Кого дожать сегодня`:
  - только `2-3` сотрудника;
  - ФИО всегда выделять жирным;
  - указывать, что именно нужно сделать сегодня и, если возможно, до какого времени.
- `Что РОП обязан сделать сегодня`:
  - ровно `3` коротких действия;
  - без воды и без повтора уже выведенных списков.
- `🔥 Молодцы за вчера`:
  - блок не считает баллы сам;
  - он берёт готовую оценку `операционная вовлечённость за вчера`;
  - показывать только сотрудников с `9+/10`;
  - ФИО всегда выделять жирным;
  - если сотрудников с `9+/10` нет, блок можно скрыть или дать нейтральную заглушку.

## Эталонный формат

```text
🎯 Фокус РОПа

Где потерян контроль:
• У <b>{manager_name_1}</b> {control_loss_signal_1}
• У <b>{manager_name_2}</b> {control_loss_signal_2}
• {control_loss_signal_3}

💸 Где взять деньги сегодня:
• <a href="{deal_url_1}">{deal_title_1}</a> — <b>{deal_amount_1}</b> — {money_action_1}
• <a href="{deal_url_2}">{deal_title_2}</a> — <b>{deal_amount_2}</b> — {money_action_2}
• <a href="{deal_url_3}">{deal_title_3}</a> — <b>{deal_amount_3}</b> — {money_action_3}

☎️ Где РОПу самому связаться с клиентом:
• <a href="{escalation_deal_url_1}">{escalation_deal_1}</a> — <b>{escalation_amount_1}</b> — {escalation_reason_1}
• <a href="{escalation_deal_url_2}">{escalation_deal_2}</a> — <b>{escalation_amount_2}</b> — {escalation_reason_2}

🕳 Упущенные возможности:
• <a href="{opportunity_deal_url_1}">{opportunity_deal_1}</a> — {opportunity_gap_1} — {opportunity_reason_1}
• <a href="{opportunity_deal_url_2}">{opportunity_deal_2}</a> — {opportunity_gap_2} — {opportunity_reason_2}

Кого дожать сегодня:
• <b>{manager_name_3}</b> — {manager_action_1}
• <b>{manager_name_4}</b> — {manager_action_2}
• <b>{manager_name_5}</b> — {manager_action_3}

Что РОП обязан сделать сегодня:
• {action_item_1}
• {action_item_2}
• {action_item_3}

🔥 Молодцы за вчера
• <b>{top_employee_1}</b> — 9.4/10 👏🔥
• <b>{top_employee_2}</b> — 9.1/10 🚀💪
```

## Если часть данных пустая

- не выводить пустой подзаголовок;
- если полезных сигналов нет, дать короткую заглушку: `• Критичных управленческих отклонений на сегодня нет`;
- если блок `🔥 Молодцы за вчера` пуст, его можно скрыть или написать: `• Лидеров с 9+/10 по операционной вовлечённости за вчера нет`.

## Если источник не подтверждён

```text
🎯 Фокус РОПа
• Ограничение: нет подтверждённых данных по задачам, встречам, движению сделок или оценке операционной вовлечённости для управленческого блока.
```
