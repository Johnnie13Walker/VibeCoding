# Источники и ограничения

## Назначение файла

Файл фиксирует допустимые источники данных, рабочую зону роли и ограничения Ларисы Ивановны. Всё, что не подтверждено, не должно подаваться как доступный live-источник.

## Приоритетные источники

- `calendar` пользователя;
- `todo` или личный task-provider;
- погодный источник по Москве;
- news source или отдельный news workflow;
- `web search`;
- локальные шаблоны, prompts и документы этого контура;
- подтверждённые shared-layer материалы Cloudbot/OpenClaw.

## Что может быть доступно уже сейчас

- локальные документы этого контура `assistant/`;
- shared integration contracts из Cloudbot legacy-контура;
- существующий `news_digest` как источник для новостного блока, если он подтверждён отдельно;
- глобальный `web search`, если он включён в runtime.

## Что нужно отдельно проверить

- подключён ли live `calendar`;
- подключён ли live `todo`;
- из какого источника берётся погода по Москве;
- каким workflow или агентом собираются новости по темам;
- подтверждён ли отдельный Telegram routing для personal assistant;
- есть ли отдельный пользовательский профиль тем для новостей;
- есть ли безопасный сценарий создания встречи в календаре.

## Что разрешено

- использовать локальные документы, шаблоны, prompts и checklist этого контура;
- проектировать отдельный `agent` и `workflow` personal assistant роли;
- использовать shared providers и workflows только как точки подключения;
- явно фиксировать ограничения, пробелы и неподтверждённые интеграции;
- держать отдельные контракты для `calendar`, `todo`, `weather`, `news`, `search`.

## Что запрещено

- CRM, сделки и sales-аналитика;
- финансы и коммерческие отчёты;
- DevOps и серверные runbooks как базовая зона роли;
- personal memory вне задач личного ассистента;
- хранение секретов, токенов и runtime-конфигов в git;
- выдача неподтверждённых live-источников за рабочие.

## Отдельный принцип

Если источник данных не подтверждён, допустимы только:

- явная пометка ограничения;
- архитектурный контракт;
- шаблон ответа;
- список того, что нужно подтвердить отдельно.
