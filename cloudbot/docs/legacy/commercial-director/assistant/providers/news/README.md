# Provider `news`

## Роль

Точка подключения к новостному источнику или shared news workflow для личного brief дня.

## Что нужно от provider

- получить новости по выбранным темам;
- вернуть `3-5` самых полезных сигналов;
- вернуть ссылки или краткие summary, если источник это поддерживает.

## Что использует provider

- workflow `personal_assistant`;
- сценарий `day_briefing`.

## Ограничения

- темы задаются отдельно и не должны смешиваться с sales-тематикой;
- новостной блок не должен дублировать общий web search;
- неподтверждённый источник не считается live.
