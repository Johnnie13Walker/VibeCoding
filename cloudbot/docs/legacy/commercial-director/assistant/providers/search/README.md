# Provider `search`

## Роль

Точка подключения к пользовательскому web search.

## Что нужно от provider

- принять свободный текстовый запрос;
- вернуть релевантные результаты;
- помочь собрать короткий ответ пользователю;
- поддержать ссылки на первоисточники, если это доступно.

## Что использует provider

- workflow `personal_assistant`;
- сценарий `search_request`.

## Ограничения

- поиск не должен подменять собой news workflow;
- ответ должен оставаться коротким и прикладным;
- неподтверждённый источник не считается live.
