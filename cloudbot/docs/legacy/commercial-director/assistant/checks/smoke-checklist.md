# Чеклист быстрой проверки роли Ларисы Ивановны

Короткий рабочий чеклист для ручной проверки качества каркаса, шаблонов и границ personal assistant контура.

- [ ] Контур Ларисы Ивановны вынесен отдельно от `commercial-director`.
- [ ] В роли явно описано, что это personal assistant, а не sales-контур.
- [ ] В `docs/larisa_assistant.md` зафиксированы границы роли.
- [ ] Есть отдельный `agent` в `agents/larisa-ivanovna/agent.ts`.
- [ ] Есть отдельные команды и workflow внутри `agents/larisa-ivanovna/`.
- [ ] Есть точки подключения к `calendar`, `tasks`, `weather`, `news`, `search`, `telegram`.
- [ ] В brief дня есть встречи, задачи, свободные окна, погода и новости.
- [ ] В brief дня время и даты указаны по `Europe/Moscow`.
- [ ] В шаблонах и prompt нет CRM, продаж, сделок и финансов.
- [ ] Сценарий добавления встречи выделен отдельно.
- [ ] Сценарий поиска выделен отдельно.
- [ ] Telegram-контур переиспользуется через существующий route Ларисы Ивановны.
- [ ] Неподтверждённые источники помечаются явно.
- [ ] В текстах нет смешения роли Ларисы Ивановны с ролью Льва Петровича.
- [ ] Формат brief дня удобен для Telegram.
