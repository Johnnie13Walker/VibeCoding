# Тесты контура Ларисы Ивановны

На этом шаге здесь хранится только каркас тестового слоя.

Что нужно добавить следующим этапом:

- contract-тесты registry команд;
- проверки guardrails по запрещенным доменам;
- тесты workflow `daily_brief` на пустой календарь, перегруженный день и недоступные providers;
- тесты Telegram-formatters;
- тесты адаптера существующего Telegram routing.
