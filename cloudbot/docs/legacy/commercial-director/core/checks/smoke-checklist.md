# Чеклист быстрой проверки роли Льва Петровича

Короткий рабочий чеклист для ручной проверки качества роли, шаблонов и отчётов.

- [ ] `Sales Copilot` зафиксирован как daily-отчёт на каждый рабочий день в `09:30` по `Europe/Moscow`.
- [ ] Сразу после `Sales Copilot` отдельным сообщением отправляется `Фокус РОПа` за тот же срез.
- [ ] Daily `Sales Copilot` везде описан как отчёт за предыдущий рабочий день.
- [ ] Еженедельный отчёт зафиксирован на пятницу `18:30` по `Europe/Moscow`.
- [ ] Weekly-отчёт везде описан как отчёт за рабочую неделю.
- [ ] `Краткая сводка` содержит только `2-4` главных сигнала и не пересказывает весь отчёт.
- [ ] `Воронка в работе` собрана единым блоком.
- [ ] Отдельного обязательного daily-блока `Деньги` рядом с `Воронкой в работе` нет.
- [ ] В `Воронке в работе` есть `Активные сделки`, `С движением за последнюю неделю`, `Без движения за последнюю неделю`, `Этап договора`, `Под риском`.
- [ ] Внутри `Под риском` есть только `Без следующего шага`, `Без коммуникации > 14 дн.`, `С просроченными задачами`.
- [ ] Детальный блок `🚨 Сделки без коммуникации > 14 дн.` не спорит с агрегатом внутри `Воронки в работе`.
- [ ] Детальный блок `⚠️ Сделки без следующего шага` не спорит с агрегатом внутри `Воронки в работе`.
- [ ] Соседние блоки не дублируют смысл.
- [ ] `Просрочки по менеджерам` вынесены отдельным блоком.
- [ ] Менеджеры с `0` просрочками не выводятся.
- [ ] В блоке просрочек есть количество, максимальная просрочка и примеры.
- [ ] В основном тексте `Sales Copilot` нет встроенного блока `Фокус РОПа`.
- [ ] `Фокус РОПа` не дублирует блок просрочек.
- [ ] `Фокус РОПа` не дублирует `Воронку в работе`, детальные риск-списки и таблицу коммуникаций.
- [ ] `Фокус РОПа` не повторяет полные списки `Без следующего шага` и `Без коммуникации > 14 дн.`.
- [ ] Из `Фокуса РОПа` убраны голые агрегаты по воронке и суммы `под риском`, если они уже есть рядом.
- [ ] В `Фокусе РОПа` есть только непустые управленческие подпункты.
- [ ] В `Фокусе РОПа` блок `Где потерян контроль` не длиннее `3-4` сигналов.
- [ ] В `Фокусе РОПа` блок `💸 Где взять деньги сегодня` содержит только сделки с суммой и действием.
- [ ] В `Фокусе РОПа` нет блока `Какие сделки нельзя упустить`.
- [ ] В `Фокусе РОПа` блок `☎️ Где РОПу самому связаться с клиентом` содержит только `1-2` сделки и поясняет, почему РОП должен сам выйти к клиенту.
- [ ] В `Фокусе РОПа` блок `🕳 Упущенные возможности` показывает провалы между обязательными этапами продажи, а не общий список зависших сделок.
- [ ] В `Фокусе РОПа` блок `Кого дожать сегодня` содержит только `2-3` сотрудника и конкретное действие.
- [ ] В `Фокусе РОПа` блок `Что РОП обязан сделать сегодня` содержит ровно `3` действия и собирает короткий план дня РОПа.
- [ ] Все ФИО менеджеров и сотрудников в `Фокусе РОПа` выделены жирным.
- [ ] Одна и та же сделка внутри `Фокуса РОПа` не кочует по подпунктам без нового управленческого смысла; третий повтор остаётся только для действительно критичной сделки.
- [ ] В блок `🔥 Молодцы за вчера` попадают только сотрудники с ровно `10/10`.
- [ ] Если сотрудников с `10/10` нет, блок `🔥 Молодцы за вчера` скрыт или заменён короткой нейтральной заглушкой.
- [ ] Длинные названия режутся аккуратно.
- [ ] Формат удобен для Telegram.
- [ ] В `Фокусе РОПа` нет пустых секций.
- [ ] В `Фокусе РОПа` нет повторов одного и того же смысла разными словами.
- [ ] В тексте нет двойных пустых строк.
- [ ] Если источник не подтверждён, это явно обозначено.
- [ ] В пользовательском тексте нет английских названий блоков и англоязычных управленческих ярлыков.
- [ ] Роль не смешана с контуром личного ассистента, инфраструктурным контуром и контуром финансового директора.
