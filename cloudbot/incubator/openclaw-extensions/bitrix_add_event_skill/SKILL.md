# Bitrix Add Event

Используй этот навык, когда пользователь просит:
- создать встречу,
- отменить встречу,
- перенести встречу (через фразу `отмени ... и создай ...`).

Источник и права:
- только `/root/.config/clawbot/calendar.env`
- только Bitrix (`BITRIX_WEBHOOK_BASE`, `BITRIX_USER_ID`)

Поддерживаемые фразы в запросе:
- `сегодня`, `завтра`, `послезавтра`
- даты: `YYYY-MM-DD`, `DD.MM[.YYYY]`, `DD <месяц> [YYYY]`
- время: `в 11:00`
- длительность: `длительностью 1 час`, `на 30 минут` (если не указана, берется 60 минут)
- название: в кавычках, либо через `название ...`
- участники: `с Петром Дудиным`, `с Петром Дудиным и Евгенией Гордиенко`
- отмена: `отмени встречу ...`
- перенос: `отмени ... и создай ...`

Команда (безопасный вызов через stdin):

```bash
set -a; . /root/.config/clawbot/calendar.env; set +a; printf '%s' "<исходный текст запроса>" | node /opt/clawdbot/scripts/bitrix-add-event.mjs --stdin
```

Правила:
- Передавай в скрипт именно исходный текст запроса пользователя без переформулировок.
- Ничего не добавляй в shell-команду кроме исходного текста в stdin.
- Если скрипт вернул ошибку (не распознаны дата/время), передай пользователю текст ошибки без изменений и попроси уточнить формат.
- Если скрипт вернул неоднозначность пользователя, задай уточняющий вопрос пользователю на основе предложенных вариантов.
- При наслоении встреч скрипт вернет предупреждение. В этом случае попроси подтверждение фразой `все правильно, создавай`.
- После успешного создания/отмены/переноса верни пользователю ответ скрипта без редактирования.
