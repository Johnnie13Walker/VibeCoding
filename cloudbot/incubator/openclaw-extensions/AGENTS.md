# AGENTS.md — Правила работы в этом репозитории

## Режим работы: только через оркестратор
1) Все операционные действия (проверки, отчеты, тестовые отправки, интеграционные прогоны) выполняются только через orchestrator workflow.
2) Прямой запуск `python`, `node`, `curl` для боевых операций запрещен, если для задачи есть workflow.
3) Если нужного workflow нет, сначала создается/обновляется workflow в orchestrator, затем выполняется запуск.
4) После каждого изменения обязательны: тесты, ручная проверка критичного сценария, проверка ошибок в логах.

## Время и язык
1) Все расписания, даты и отчеты ведутся в Europe/Moscow.
2) Вся коммуникация и сообщения пользователю — на русском языке.
