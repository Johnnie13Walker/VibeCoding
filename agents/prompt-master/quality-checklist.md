# prompt-master: quality checklist

## Checklist входа

- [ ] Понятно, что пользователь хочет.
- [ ] Понятна бизнес-цель.
- [ ] Понятен конечный артефакт.
- [ ] Определён домен задачи.
- [ ] Определены недостающие данные.
- [ ] Определены риски.
- [ ] Проверена релевантная память workspace.

## Checklist декомпозиции

- [ ] Задача разбита на этапы.
- [ ] Указаны зависимости между этапами.
- [ ] Понятно, что можно делать параллельно.
- [ ] Понятно, что должно идти последовательно.
- [ ] Нет лишних агентов.
- [ ] Для write-задач указан file ownership.
- [ ] File ownership конкретен: перечислены точные файлы или модули, без формулировок «и связанные файлы».
- [ ] File ownership не пересекается с активным ownership других задач или агентов.
- [ ] Все агенты, кроме `Мастера`, помечены как read-only для этой задачи.

## Checklist маршрутизации

- [ ] Выбран основной агент.
- [ ] Выбраны проверяющие агенты.
- [ ] `Справочник` подключён, если нужны официальные источники.
- [ ] `Ревизор` подключён для high-risk задач.
- [ ] `Испытатель` подключён для финальной проверки.
- [ ] Для визуальных задач есть `Продуктолог + Дизайнер` до реализации и `Испытатель + Дизайнер` после реализации.

## Checklist production-ready prompt

- [ ] Есть контекст.
- [ ] Есть цель.
- [ ] Есть бизнес-смысл.
- [ ] Есть входные данные.
- [ ] Есть ограничения.
- [ ] Есть ожидаемый результат.
- [ ] Есть критерии качества.
- [ ] Есть формат ответа.
- [ ] Есть риски.
- [ ] Есть технические детали.
- [ ] Есть условия остановки.

## Checklist orchestration

- [ ] Есть flow выполнения.
- [ ] Есть prompts для каждого агента.
- [ ] Есть context handoff.
- [ ] Есть aggregation logic.
- [ ] Есть acceptance checklist.
- [ ] Пользователь не получает сырой черновик вместо финального результата.

## Итоговый статус

Использовать один из статусов:

- `ОК` — prompt и orchestration готовы к запуску.
- `Нужны данные` — нельзя безопасно запускать без уточнений.
- `Есть риски` — можно запускать только после ревизии или ограничений.
- `Не готово` — задача слишком неоднозначна или нарушает ограничения.
